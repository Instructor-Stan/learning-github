Cloning a repository using gitbash.

  • Change to a high level directory where your git projects with be stored (ie. "cd" - with no paramter, puts you in your home directory.
  • Type "git init" to setup the files Git will need to manage files on your PC.
  • Type "git clone git@github.com:StanCrump/learning-github.git" to clone this repository. The clone command can be found on the web under the "Code" button in the repository you want to clone.
  • Type "cd <directory> where directory is the part of the name you cloned between the last slash (/) and ".git" on the end. Ex. learning-github
  • Type "git status" to confirm git knows about your files.
  • Type "code ." to start VS Code editor and open the directory you just cloned. (Now change some code.)
  • Type "git status" to see that git has acknowledged your changes and git prompts you to the next step(s).
